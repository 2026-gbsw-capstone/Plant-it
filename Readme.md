# 🌱 Plant it

## 반려식물을 더 오래, 더 건강하게 돌보기 위한 올인원 관리 앱

반려동물을 돌보듯, 식물 역시 꾸준한 관심과 관리가 필요합니다.  
하지만 바쁜 일상 속에서는 물 주는 시기를 놓치거나, 성장 과정을 기록하지 못하거나, 상태 변화를 뒤늦게 알아차리는 일이 자주 발생합니다.

**Plant it**은 이러한 불편을 해결하기 위해 기획한 **반려식물 관리 모바일 앱**입니다.  
사용자는 식물을 등록하고, 관리 주기를 확인하며, 성장 기록을 남기고, 필요한 시점에 알림을 받을 수 있습니다.

단순한 기록 앱을 넘어,  
식물도감 · 커뮤니티 · AI 기반 분석 기능까지 확장 가능한 구조를 통해  
**반려식물 라이프 전반을 돕는 서비스**를 목표로 합니다.

## 🧭 핵심 기획 방향

Plant it은 식물별 관리 주기를 기반으로  
사용자에게 **개별화된 관리 경험**을 제공하는 것을 핵심으로 합니다.

- 식물별 상태 및 관리 정보를 한눈에 확인
- 반복적인 관리 일정을 놓치지 않도록 알림 제공
- 사진과 메모 기반의 성장 기록 기능
- 정보 탐색 및 커뮤니티 확장 고려

단순 기능 나열이 아닌,  
**사용자의 관리 루틴을 돕는 흐름 중심 서비스**로 설계했습니다.

## 🚀 주요 기능

### 🌿 식물 등록 및 관리 현황
- 반려식물 등록 및 기본 정보 관리
- 식물별 상태 및 최근 관리 내역 확인

### ⏰ 관리 주기 기반 일정
- 물주기, 비료 주기 등 관리 일정 자동 계산
- 식물별 맞춤 관리 경험 제공

### 🔔 알림 기능
- 푸시 알림 / 로컬 알림을 통한 관리 루틴 지원
- 관리 시기 놓침 방지

### 📸 성장 기록
- 사진 + 메모 기반 성장 과정 기록
- 시간 흐름에 따른 변화 확인

### 📚 식물도감
- 식물별 특징 및 관리 팁 제공
- 앱 내 정보 탐색 경험 강화

### 💬 커뮤니티
- 게시글 / 댓글 / 좋아요 / 신고 기능
- 사용자 간 정보 공유

### 🤖 AI 기능 확장
- 식물 종 식별
- 건강 상태 분석
- 상담형 관리 기능
